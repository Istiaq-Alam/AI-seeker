FAQ

Quick answers to common questions.

What does this script do? ▾

It automates the setup process for Claude Code on a Linux machine. Use the 3-step install or the one-liner.

Is it really offline? ▾

The flow is offline-friendly in the sense that it’s a single script you can download and reuse. If the script downloads dependencies, you’ll still need network access for those parts.

Copy buttons don’t work? ▾

Some browsers block clipboard access on file:// pages. Serve the page (or open it over http/https) and try again.
